What is KubeWharf?


KubeWharf is an open source orgnization from ByteDance built on Kubernetes.
It aims to supply open source solution to address key challenges of large-scale cloudnative situations,
including scalability, stability, multi-tenancy isolation, resource efficiency and obersavability.
